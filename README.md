<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/WGoodsonDev/ssbm-combo-visualizer.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
